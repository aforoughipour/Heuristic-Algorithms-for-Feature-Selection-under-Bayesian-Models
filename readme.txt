obf_gaussian is optimal Bayesian filter assuming Gaussian features.
TwoMNC_Robust is 2MNC_Robust assuming Gaussian features.
REMAIN_gaussian is REMAIN for Gaussian features.
POFAC_gaussian is POFAC assuming Gaussian features.
SPM_gaussian is SPM assuming Gaussian features.

Refer to each code for details how to run them, and refer to the following paper
for details about each algorithm:

Heuristic Algorithms for Feature Selection under Bayesian Models with Block-diagonal Covariance Structure
by A Foroughipour and L A Dalton
